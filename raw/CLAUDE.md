# raw/ — 원본 (Raw Sources)

> 영균이 수집한 **불변 원본**. AI는 여기서 **읽기만** 하고 **절대 수정·삭제하지 않는다**.
> 이곳이 진실의 출처(source of truth)다.

## 인풋 유형별 하위 폴더

### 웹 클리핑 (Obsidian Web Clipper)

저장소 루트의 `clipper-*.json` 템플릿으로 클리핑하면 아래 폴더에 자동 저장된다.

| 폴더        | 담는 것                  | 클리퍼 템플릿           |
| ----------- | ------------------------ | ----------------------- |
| `articles/` | 웹 글·공식 문서·블로그   | `clipper-article.json`  |
| `youtube/`  | 유튜브 영상 (설명·자막)  | `clipper-youtube.json`  |
| `podcasts/` | 팟캐스트 에피소드        | `clipper-podcast.json`  |
| `books/`    | 책 (소개·발췌·챕터 메모) | `clipper-book.json`     |
| `research/` | 논문·연구 자료           | `clipper-research.json` |

### 직접 작성 입력

| 폴더         | 담는 것                                     |
| ------------ | ------------------------------------------- |
| `lectures/`  | 부트캠프 강의·수업 노트, 강의 자료          |
| `practice/`  | 직접 한 실습 — 코드, 실험 기록, 디버깅 로그 |
| `notes/`     | 기존 노션·GitHub에서 옮겨온 필기            |
| `assets/`    | 이미지·다이어그램 등 첨부 파일              |
| `dialogues/` | AI 채팅 기록                                |

## 클리핑 원본의 frontmatter 스키마

웹 클리퍼로 수집한 원본은 아래 YAML frontmatter를 갖는다. **Ingest는 이 필드를 읽어 위키로 소화한다.**

| 필드                                | 의미                                            | 인제스트 시 매핑            |
| ----------------------------------- | ----------------------------------------------- | --------------------------- |
| `source_type`                       | `article`/`youtube`/`podcast`/`book`/`research` | 소스 종류 판별              |
| `title`                             | 제목                                            | `wiki/sources/` 페이지 제목 |
| `url`                               | 원본 링크                                       | 위키 인용 출처              |
| `author`/`authors`/`channel`/`host` | 작성자·채널·진행자                              | 소스 페이지 메타            |
| `published`                         | 원본 발행일                                     | 소스 페이지 메타            |
| `clipped`                           | 클리핑한 날짜                                   | `log.md` 기록 참고          |
| `description`                       | 요약·초록                                       | 소스 요약의 출발점          |
| `tags`                              | `clippings/<type>` 네스티드 태그                | `wiki` 태그로 승계          |

본문에는 `## 💭 내 메모` 섹션이 있다 — 영균이 직접 적은 메모이므로 **인제스트 시 가장 우선해서 반영**한다 (왜 저장했는지가 곧 그의 맥락이다).

## AI 규칙

- `raw/`의 파일은 **수정 금지**. 통합·요약·연결은 모두 `wiki/`에서 한다.
  인제스트 진행 상태 추적도 raw 파일이 아니라 `wiki/log.md`·`wiki/index.md`에서 한다.
- 새 원본이 들어오면 **Ingest** 작업을 수행한다 (루트 `CLAUDE.md` 참조).
- 원본을 위키에서 인용할 때는 `raw/<폴더>/<파일명>` 경로로 출처를 명시한다.
- 마크다운에 인라인 이미지가 있으면, 본문 텍스트를 먼저 읽고 필요한 이미지를 따로 확인한다.
