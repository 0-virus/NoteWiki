# Output/ — 결과물

> AI가 영균에게 전달하는 **최종 산출물**을 모으는 곳.

## 담는 것

- 위키 내용을 바탕으로 만든 깔끔한 **필기** (md / html)
- 비교표, 분석 문서, 슬라이드(Marp), 다이어그램
- 면접 대비 자료, 프로젝트 적용 가이드 등

## AI 규칙

- 결과물은 `wiki/`의 페이지들을 **종합·재구성**해 만든다. 원본 분석이 필요하면 먼저 `wiki/`에 ingest한 뒤 결과물을 만든다.
- 출력 형태: 기본 **Markdown**, 필요 시 HTML. 비전공자가 이해하도록 풀어 쓰되 원리는 생략 X.
- 결과물이 재사용할 만한 지식(새 비교·분석·연결)을 담았다면, `wiki/`에도 페이지로 **환류**한다.
- 파일명에 날짜나 주제를 넣어 나중에 찾기 쉽게 한다 (예: `2026-05-18-was-정리.md`).
